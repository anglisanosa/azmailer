Introduction
============


AzMailer is a Python class designed to facilitate sending emails using Azure Communication Services.

Inspired by the Azure Communication Services documentation `here <https://learn.microsoft.com/en-us/azure/communication-services/quickstarts/email/send-email?tabs=linux%2Cconnection-string&pivots=programming-language-python>`_, AzMailer simplifies the process of sending emails by providing an easy-to-use interface for constructing and sending email messages.
